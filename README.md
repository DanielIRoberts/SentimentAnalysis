# SentimentAnalysis
Impliments Sklearn's sentiment analysis

General Info-----------------------------------------------------------------------------------------

Name	- Daniel Roberts	

Assignment: Sentiment Analysis
IDE	- Visual Studio Code

Included Files---------------------------------------------------------------------------------------

SentimentAnalysis.py
	- Reads in and preprocesses the data
	- Fits a MultinomialNB and predicts sentiment
	- Loops through modeling until user is satisfied
	- Finds airline sentiment averages
	- Finds airline with best sentiment average
	- Outputs plot of alpha values with accuracy

How to Run-------------------------------------------------------------------------------------------

Takes command line arguemnt for file location
Takes user input for model variable
Loops through models until user input is n

Example command line -
	python SentimentAnalysis.py Tweets.csv
	Enter value for alpha or n to quit:
	1
	Enter value for alpha or n to quit:
	n
